# TownSorveteria
O repositório consiste em um algoritmo para armazenar, ler e exibir os dados referentes a pesquisa de satisfação de uma sorveteria. <br>
Utilizando linq, filtra e encontra o id mais votado dentre as 5 opções predefinidas e, caso ocorra um empate, o menor id será considerado o mais popular.

# Pré Requisitos
A solução foi desenvolvida em torno da plataforma .net, mais especificamente, em sua versão 5. Portanto é necessário possuir o runtime que pode ser encontrado no link abaixo.
  * [dotnet.microsoft.com](https://dotnet.microsoft.com/en-us/download/dotnet/5.0).
